# Machines Claude Plugin

Use Machines in Claude to check balances, manage cards, handle deposits, and safely reveal card details when needed.

## Install

In Claude Code:

```text
/plugin marketplace add https://github.com/machines-cash/machines-claude-plugin
/plugin install machines@machines-plugins
```

## Docs

- https://docs.machines.cash/claude-plugin

## Validate

```bash
claude plugin validate .
claude plugin validate ./packages/claude-plugin
```
